# Cramly

Cramly is a web-based study and canvas application designed for fast, keyboard-driven workflows. It combines note-taking, visual mapping, drawing tools, and study modes into a single interactive environment.

---

## Overview

Cramly helps users:

- Organize ideas visually on a canvas  
- Create and manage notes dynamically  
- Connect elements for better understanding  
- Study using interactive flashcards  
- Navigate efficiently using keyboard shortcuts  

The system focuses on speed, usability, and minimal UI friction.

## Features

### Keyboard-Driven Workflow
- Full application control using keyboard shortcuts  
- Context-aware input handling (disabled while typing)  
- Fast switching between tools and modes  

### Canvas System
- Add and edit text elements  
- Insert images and visual content  
- Drag, move, and organize elements freely  

### Drawing Mode
- Freehand drawing on canvas  
- Stroke-based undo system  

### Smart Selection
- Single and multi-element selection  
- Grouping functionality  
- Lasso selection tool  

### Study Mode
- Flashcard-based learning system  
- Flip cards and navigate using arrow keys  

### Panels and Tools
- Deck and note management  
- Shape picker  
- Sticker and meme panel  
- Flow and connection tools

## Keyboard Shortcuts

### Commands (Ctrl / Cmd)

| Shortcut | Action |
|----------|--------|
| Ctrl + D | Open Deck Modal |
| Ctrl + N | Open Note Modal |
| Ctrl + . | Toggle Dark Mode |
| Ctrl + G | Toggle Snap |

---

### Tools

| Key | Action |
|-----|--------|
| T | Add Text Box |
| S | Toggle Shape Picker |
| I | Insert Image |
| C | Drop Category |
| M | Open Meme Panel |
| D | Toggle Deck Panel |
| A | Toggle Connect Mode |
| P | Toggle Draw Mode |
| K | Toggle Sticker Panel |
| F | Flip Card / Toggle Flow Panel |
| L | Toggle Lasso Tool |
| G | Group Selected Elements |
| B | Save Bookmark |

---

### Navigation

| Key | Action |
|-----|--------|
| / | Open Search |
| \ | Toggle Dock |
| + / = | Zoom In |
| - | Zoom Out |

---

### Study Mode

| Key | Action |
|-----|--------|
| Arrow Right | Next Card |
| Arrow Left | Previous Card |

---

### Editing

| Key | Action |
|-----|--------|
| Delete / Backspace | Remove selected element(s) |
